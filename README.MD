# parcialEstructuraCorte2

Proyecto de estructuras de datos en JavaScript — Corte 2.

## 📚 Contenido

### 🔺 Pila (`pila.js`)
Simula una pila (LIFO) con elementos de tipo objeto.
- `agregar(objeto)` — Agrega un objeto al tope de la pila
- `eliminar()` — Elimina el elemento del tope

**Ejemplo de objeto:**
```json
{
  "Nombre": "Orlando",
  "Edad": 30
}
```
(usar ¡[ ](url) )
### 🔷 Cola (`cola.js`)
Simula una cola (FIFO) con datos numéricos.
- `agregar(numero)` — Agrega un número al final de la cola
- `eliminar()` — Elimina el número del frente
